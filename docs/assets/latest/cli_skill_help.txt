Run structured diagnostic skills (shared with Web Agent)

Usage: probing-cli skill [OPTIONS] <COMMAND>

Commands:
  list     List available diagnostic skills
  install  Install skills into Cursor, Claude Code, and Codex skill directories
  update   Update installed skills from bundled source
  run      Run a diagnostic skill against the target process
  help     Print this message or the help of the given subcommand(s)

Options:
  -v, --verbose  Enable verbose mode
  -h, --help     Print help
